## 🚀 模型訓練程式 (Model Training)

本專案包含一個用於辨識抽籤號碼的電腦視覺模型。為了加速訓練流程，建議使用 Google Colab 的 GPU 資源：

1. **下載檔案：** 將本儲存庫中的訓練程式碼（`.ipynb` 或 `.py`）與 [Google 雲端資料集](https://drive.google.com/drive/folders/1fNZDlP7a8Sn76PZJPTEKPFXgvYxthApA?usp=drive_link) 下載至本地。
2. **上傳至 Colab：** 將檔案上傳至您的 Google Drive 或直接匯入 Google Colab。
3. **開啟 GPU 加速：** 於 Colab 選單選擇 `執行階段` > `變更執行階段類型` > 貼心切換為 `GPU`。
4. **開始訓練：** 依序執行儲存格，即可快速完成籤號視覺辨識模型的訓練。
